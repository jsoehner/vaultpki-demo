How the Demo Works

This demo leverages HashiCorp Vault as a private CA to issue short-lived certificates dynamically.

  • Vault: Root and Intermediate CA mounts manage cryptographic trust keys.
  • Vault Agent: Periodically authenticates using AppRole, monitors templates, and requests new certificates with a 1-minute TTL.
  • NGINX: Serves secure traffic on port 8443. Upon rotation, Vault Agent signals NGINX (via SIGHUP) to reload certificates with zero downtime.
